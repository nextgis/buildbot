# Intro

Windows 7 qemu/kvm worker on Ubuntu 16.04 LTS Server.

# Prepare

For Ubuntu 16.04 LTS host server

main instructions: https://github.com/kholia/OSX-KVM
get seabios 1.10 here: https://launchpad.net/~alindt/+archive/ubuntu/xenial-backports
get qemu 2.10 here: https://launchpad.net/~alindt/+archive/ubuntu/xenial-backports

Need at least qemu 2.9+
